# 💱 Currency Converter

A simple and responsive Currency Converter built using HTML, CSS, and JavaScript. It allows users to convert currencies in real-time using exchange rate data and provides an intuitive user interface with country flags.

## 📸 Preview

Convert currencies instantly by selecting source and target currencies and entering the desired amount.

### Features

- 🌍 Supports multiple currencies
- 🔄 Swap currencies instantly
- 🚩 Country flags for better user experience
- 📈 Real-time exchange rates
- 📱 Responsive design
- ⚡ Fast and lightweight

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Exchange Rate API

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/currency-converter-html-css-js.git
```

2. Navigate to the project folder

```bash
cd currency-converter-html-css-js
```

3. Open `index.html` in your browser.

## 💡 How It Works

1. Enter the amount to convert.
2. Select the source currency.
3. Select the target currency.
4. Click **Get Exchange Rate**.
5. View the converted amount instantly.
